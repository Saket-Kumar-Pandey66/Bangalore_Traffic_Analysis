# Dashboard Instructions: Bangalore Traffic Analysis

## Tableau Dashboard (Recommended)

### Prerequisites
- Download and install [Tableau Public](https://public.tableau.com/) (free) or Tableau Desktop.
- Ensure `outputs/data_for_r.csv` has been generated by running `python_analysis.py`.

---

### Step-by-Step Guide

#### Step 1: Connect to Data
1. Open Tableau.
2. Click **Connect → Text file** and select `outputs/data_for_r.csv`.
3. Tableau will auto-detect column types. Verify:
   - `Date` → Date
   - `Traffic_Volume`, `Congestion_Level`, etc. → Number (decimal)
   - `Area_Name`, `Weather_Conditions` → String

#### Step 2: Create "Traffic by Hour/Day" Sheet
1. Drag `Day_of_Week` to **Columns**.
2. Drag `Traffic_Volume` to **Rows** → change to AVG.
3. Drag `Congestion_Level` to **Color** (Marks card) → change to AVG.
4. Change chart type to **Bar**.
5. Sort by descending traffic volume.
6. Title: *"Average Traffic Volume by Day of Week"*.

#### Step 3: Create "Top Congested Locations" Sheet
1. Drag `Area_Name` to **Rows**.
2. Drag `Congestion_Level` to **Columns** → change to AVG.
3. Sort descending by congestion.
4. Drag `Road_Name` as a detail.
5. Add color by `Congestion_Level` (gradient: green → red).
6. Title: *"Top Congested Areas in Bangalore"*.

#### Step 4: Create "Heatmap" Sheet
1. Drag `Month` to **Columns**, `Day_of_Week` to **Rows**.
2. Drag `Congestion_Level` to **Color** → change to AVG.
3. Change chart type to **Square** or **Density**.
4. Use a Red-Yellow-Green diverging color palette.
5. Title: *"Congestion Heatmap: Month × Day of Week"*.

#### Step 5: Create "Time Series" Sheet
1. Drag `Date` to **Columns** (ensure it's set to Day level).
2. Drag `Traffic_Volume` to **Rows** → change to AVG.
3. Add a trend line (Analytics tab → Trend Line).
4. Add `Congestion_Level` as a dual axis.
5. Title: *"Traffic Volume and Congestion Over Time"*.

#### Step 6: Create "Weather Impact" Sheet
1. Drag `Weather_Conditions` to **Rows**.
2. Drag `Congestion_Level` to **Columns** → AVG.
3. Add `Incident_Reports` as SUM to size.
4. Title: *"Weather Impact on Traffic"*.

#### Step 7: Assemble Dashboard
1. Click **New Dashboard** tab.
2. Set size to **Automatic** or **1200 × 800**.
3. Drag all sheets onto the dashboard canvas.
4. Add **filters** for Area_Name, Weather_Conditions, Day_of_Week.
5. Enable **filter actions** so clicking one chart filters others.
6. Add a title: *"Bangalore City Traffic Analysis Dashboard"*.

#### Step 8: Publish
1. Go to **Server → Tableau Public → Save to Tableau Public**.
2. Create account if needed and publish.
3. Share the URL.

## Power BI Dashboard (Alternative)

### Prerequisites
- Download [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free, Windows only).

### Step-by-Step Guide

#### Step 1: Import Data
1. Open Power BI Desktop.
2. Click **Get Data → Text/CSV**.
3. Select `outputs/data_for_r.csv`.
4. Click **Transform Data** to open Power Query Editor.
5. Verify column types and click **Close & Apply**.

#### Step 2: Create Visuals
Add these visuals to a new report page:

| Visual Type          | X-Axis / Category | Value           | Title                              |
|----------------------|--------------------|-----------------|-------------------------------------|
| Clustered Bar Chart  | Day_of_Week        | AVG Traffic_Volume | Traffic by Day of Week            |
| Stacked Bar Chart    | Area_Name          | AVG Congestion_Level | Top Congested Areas             |
| Matrix / Heatmap     | Month × Day_of_Week| AVG Congestion_Level | Congestion Heatmap              |
| Line Chart           | Date               | AVG Traffic_Volume  | Traffic Volume Time Series       |
| Donut Chart          | Weather_Conditions | COUNT records       | Weather Distribution             |

#### Step 3: Add Slicers (Filters)
- Add **Slicer** visuals for: Area_Name, Weather_Conditions, Is_Weekend.

#### Step 4: Format
1. Apply a dark theme: View → Themes → Select a dark theme.
2. Add a text box with title: *"Bangalore Traffic Analysis Dashboard"*.
3. Align and resize all visuals.

#### Step 5: Publish
1. Click **Publish** → sign in to Power BI service.
2. Select a workspace and publish.

- **Average Traffic Volume**: Overall and by area
